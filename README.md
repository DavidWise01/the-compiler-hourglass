> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **LOGISMÓS**.

THE HOURGLASS (purple papers VI) — a compile has a SHAPE: code climbs from flat text up into meaning (text → tokens → tree → IR, a clean funnel), squeezes through one narrow universal NECK (the IR), then descends into a specific machine (IR → optimize → assembly → micro-ops, a lumpy gradient walk). You climb a TREE and descend a GRAPH, and the shape changes exactly at the neck. THE NARROW-WAIST PRINCIPLE: one thin middle kills the combinatorial explosion (N+M front/back-ends instead of N×M whole compilers — why LLVM won) — the same waist as the internet (IP) and the cell (the genetic code). LIVE: languages pour in the top, machines catch at the bottom, programs bloom at the load-bearing 0.
